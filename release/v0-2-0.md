# v0.2.0

Release date: 2026-04-20

## Summary

- Public API ปรับให้เรียบง่ายขึ้น โดยเน้น `Controller` และ `Repository` เป็น extension points หลัก
- ปรับ subpath exports ให้เหลือชุดใช้งานจริง: `controller`, `repository`, `entities`
- รวม field helpers ไว้ภายใต้ `entities` เพื่อให้ naming สอดคล้องกับโดเมน
- ปรับโครงสร้างเอกสารให้สอดคล้องกับการใช้งานจริงของ v0.2.0

## Import

```ts
import { Controller, Repository } from '@aps/next-api'
import Controller from '@aps/next-api/controller'
import Repository from '@aps/next-api/repository'
import { BaseEntity, StringField } from '@aps/next-api/entities'
```

## Breaking Changes

- Removed `@aps/next-api/base-controller` export
- Removed `@aps/next-api/base-repository` export
- Removed `BaseController` and `BaseRepository` named exports from root entrypoint
- Replaced legacy field import path (`@aps/next-api/field`) with `@aps/next-api/entities`

## Migration Notes

- Replace `BaseController` with `Controller`
- Replace `BaseRepository` with `Repository`
- Update field helper imports:

```ts
// before
import { BaseEntity, StringField } from '@aps/next-api/field'

// after
import { BaseEntity, StringField } from '@aps/next-api/entities'
```

## Exports (v0.2.0)

- `@aps/next-api`
- `@aps/next-api/controller`
- `@aps/next-api/repository`
- `@aps/next-api/entities`
